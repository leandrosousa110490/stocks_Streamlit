# Stock Streamlit App

A stock market analysis dashboard built with Streamlit, DuckDB, and YFinance.

## Features
- **Dashboard**: Search for stocks, view charts with trend lines.
- **Market Explorer**: View top stocks by Market (NASDAQ, FTSE, etc.) or Sector.
- **My Portfolio**: Track your favorite stocks with custom performance metrics.
- **Custom Analysis**: Calculate averages and percentage changes over flexible timeframes (1 Week, 1 Year, etc.).

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```
